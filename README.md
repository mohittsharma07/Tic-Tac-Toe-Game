# 🎮 Tic Tac Toe Game

A modern and responsive Tic Tac Toe game built using **HTML, CSS, and JavaScript**.

## 🚀 Features

* 👥 Player vs Player Mode
* 🤖 Player vs Computer Mode
* 📱 Fully Responsive Design
* 🔄 Reset Game Option
* 🆕 New Game Option
* 🏆 Winner Detection
* 🤝 Tie Detection
* ⚡ Fast and Lightweight

---

## 📸 Preview

The game provides two playing modes:

### Player vs Player

Two users can play against each other on the same device.

### Player vs Computer

A single player competes against the computer. The computer automatically makes its move after the player's turn.

## 🎯 How It Works

### 1. Game Start

When the page loads, an empty 3×3 board is displayed.

### 2. Select Mode

Users can choose:

* Player vs Player
* Player vs Computer

### 3. Taking Turns

* Player O always starts first.
* Players click on an empty box to place their symbol.
* Once a box is selected, it cannot be changed.

### 4. Computer Mode

When Player vs Computer mode is selected:

* The player uses "O".
* The computer uses "X".
* After the player moves, the computer automatically chooses an available box.
